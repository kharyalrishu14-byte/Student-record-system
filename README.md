# Student Record System

A web-based Student Record System built using Flask.

# Features

- Student can login if already register otherwise register First 
- See their details like (name, email, date of birth, course, gender etc)
- Update Student records
- Delete Student 
- Logout their Id's

# Technologies Used
- Pyhton
- SQL
- Flask
- HTML
- CSS
# How to Run
1. Clone the repository:
   git clone https://github.com/kharyalrishu14-byte/Student-record-system.git

2. Go to project folder:
   cd Student-record-system

3. Create virtual environment:
   python -m venv env

4. Activate environment:
   env\Scripts\activate   (Windows)

5. Install dependencies:
   pip install -r requirements.txt

6. Run the app:
   python app.py
